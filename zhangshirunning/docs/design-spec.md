# 跑酷小游戏 — 设计规范

> 版本：v1.0
> 最后更新：2026-07-22

---

## 1. 整体美术风格

**关键词**：色彩鲜艳、卡通、活泼、明快

- 使用明亮饱和的色调
- 圆润的造型，避免尖锐线条
- 角色和物品使用简单几何图形绘制
- 背景采用扁平化卡通风格

## 3. 三层视差背景规范

```
┌──────────────────────────────┐
│     天空层（滚动最慢）        │  ← 蓝色渐变 + 白色云朵
│   ☁️      ☁️    ☁️           │
├──────────────────────────────┤
│     远景层（滚动中等）        │  ← 远山/建筑剪影
│   ⛰️⛰️   🏠🏠   ⛰️⛰️        │
├──────────────────────────────┤
│     近景层（滚动最快）        │  ← 地面 + 草地
│  🌱🌱🌱🌱🌱🌱🌱🌱🌱🌱🌱🌱   │
│  ━━━━━━━━━━━━━━━━━━━━━━━━   │
└──────────────────────────────┘
```

### 滚动速度比
- 天空层：0.5 倍基础速度
- 远景层：1 倍基础速度
- 近景层（地面）：1 倍基础速度（与障碍物同步）

## 4. 角色外观规范

### 4.1 身体比例（以头部为 1 单位）
```
    O        ← 头部（圆，直径 20px，含脸部颜色）
   /|\       ← 身体 + 手臂（黄衣，高约 30px）
   / \       ← 腿部（绿裤，高约 25px）
```

角色整体高度约 60px，站立在画面 y=380 处（地面线）。

### 4.2 奔跑动画
- 双腿交替前后摆动（正弦波驱动的旋转）
- 手臂小幅度交替摆动
- 动画周期：约 0.3 秒

### 4.3 跳跃动画
- 身体整体提升，最高点约 y=260（地面线 - 120px）
- 腿部收起（类似跳起姿势）
- 抛物线轨迹

### 4.4 低头动画
- 角色垂直缩放：高度缩小到约 60%
- 持续 0.5 秒后恢复
- 视觉上表现为蹲下/滑铲

## 5. 心率脸色变化规范

心率值范围为 0~100（归一化 0~1.0）：

| 心率 % | 脸部颜色 | 色值 |
|--------|----------|------|
| 0-25% | 正常肤色 | `#FFDAB9` |
| 25-50% | 浅紫色 | `#DDA0DD` |
| 50-75% | 中紫色 | `#9932CC` |
| 75-100% | 深紫色 | `#4B0082` |

使用颜色插值实现平滑过渡（`lerpColor`），而非分段切换。

## 6. 障碍物外观规范

### 6.1 地面障碍物（箱子）
- 矩形造型，约 40×40 px
- 棕色主体 `#8B4513`，深色边框
- 可选：画十字封箱带

### 6.2 上方障碍物（横梁）
- 横向矩形，约 100×20 px
- 金属灰 `#808080` 主体
- 黄色警示条纹
- 悬浮高度：角色站立时头部上方的位置（y ≈ 300）

---

## 7. 道具外观规范

### 7.1 药丸
- 红白双色胶囊造型
- 约 16×24 px
- 红色 `#FF0000` + 白色 `#FFFFFF` 各半

### 7.2 巧乐兹
- 冰棍/雪糕造型
- 约 14×30 px
- 粉色雪糕体 + 棕色巧克力涂层 + 木棍

### 7.3 雪碧
- 绿色汽水瓶造型
- 约 16×28 px
- 绿色瓶身 + 白色标签 + 银色瓶盖

## 8. UI 组件规范

### 8.1 分数
- 位置：左上角 (20, 30)
- 字体：24px bold，白色，深色描边
- 格式：`分数: 12345`

### 8.2 心率条
- 位置：画布顶部居中，(250, 10)，宽 400px，高 20px
- 左侧 ❤ 心形图标
- 条形进度条，颜色随心率值变化：绿色→黄色→红色
- 圆角矩形

### 8.3 能量条
- 位置：心率条正下方，(250, 36)，宽 400px，高 10px
- 仅在加速激活时显示
- 蓝色 `#2196F3`
- 显示加速剩余时间比例

### 8.4 开始界面
- 游戏标题：居中，48px bold 白色文字
- "按空格键开始"：居中，24px 白色文字，闪烁动画
- 操作说明：下方列出空格=M 键的功能

### 8.5 结束界面
- "游戏结束"：居中，40px bold 红色文字
- 本局分数：居中，28px 白色文字
- 历史最高分：居中，28px 金色文字（如果是新纪录则显示 🆕）
- "按空格键重新开始"：居中，20px 白色闪烁文字

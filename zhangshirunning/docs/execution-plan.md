---

## 执行原则

1. **每次只推进一个阶段**，完成后验收，确认无误再进入下一阶段
2. **每阶段结束更新开发日志**到 `dev-logs/` 文件夹
3. **所有代码在单个 `index.html` 文件中**，逐步叠加功能
4. **每个阶段独立可测试**，不依赖后续阶段的功能

## 阶段 4：跳跃 + 低头操作

**目标**：空格跳跃、M 低头，动画流畅

### 任务清单
- [x] 空格键触发跳跃：抛物线轨迹（初速度 -12 + 重力 0.6），最高约 120px，约 40 帧完成
- [x] M 键触发低头：角色高度缩小到 55%，持续 30 帧（0.5 秒）后自动恢复
- [x] 防连发：空中不能跳跃，低头期间不能跳跃，空中不能低头
- [x] 左上角状态调试面板显示跳跃高度 / 低头剩余帧数

### 验收标准
- 按空格：角色平滑跳跃，落地后才能再跳
- 按 M：角色低头 0.5 秒后恢复
- 空中按空格无效（无二段跳）
- 操作响应即时，无延迟
